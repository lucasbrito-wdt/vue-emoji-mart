# Vue Emoji Mart Vite Demo

Aplicação de demonstração mínima para testar o componente `<Picker>` do vue-emoji-mart.

## Instalação

```bash
pnpm install
```

## Desenvolvimento

```bash
pnpm dev
```

Abre automaticamente `http://localhost:5173` no navegador.

## Build

```bash
pnpm build
```

Gera os arquivos otimizados em `dist/`.

## Estrutura

- `src/App.vue` — componente raiz com Picker integrado
- `src/main.ts` — entry point do Vite
- `index.html` — template HTML
- `vite.config.ts` — configuração do Vite
- `tsconfig.json` — configuração TypeScript

## Notas

Esta é uma estrutura mínima para QA manual. Não é um projeto completo de produção.
